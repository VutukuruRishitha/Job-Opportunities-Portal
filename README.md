# TaskFlow - Job Opportunities Portal

## Overview
This project is a responsive frontend landing page and job opportunities portal built for TaskFlow. It features a modern design, dark mode functionality, client-side search and filtering, and a mock job API integration using Vanilla HTML5, CSS3, and JavaScript (ES6).

## Features
- **Responsive Layout**: Designed to adapt across mobile, tablet, and desktop viewports using Flexbox and CSS Grid.
- **Dark Mode**: Toggleable dark theme with state persistence via `localStorage`.
- **Job Listings with Filtering**: Browse job opportunities with real-time filtering by keyword, location, and company.
- **Pagination**: Client-side pagination logic to manage job listings.
- **Skeleton Loaders**: Visual shimmering UI elements displayed while data is simulated to load.
- **Form Validation**: Client-side form validation for the contact section.

## Technology Stack
- HTML5
- CSS3 (Custom Variables, Flexbox, Grid)
- JavaScript (Vanilla ES6)
- FontAwesome (Icons)

## Setup Instructions
1. Clone this repository or download the source code.
2. Navigate to the `frontend-task` directory.
3. Open `index.html` directly in any modern web browser.
4. Alternatively, use a local development server (e.g., `npx serve` or VS Code Live Server) for the best experience.
